# 🎮 UML Maze: System Guardians

**UML Maze: System Guardians** is a cyberpunk-themed, browser-based educational maze game that combines software engineering concepts with classic arcade-style gameplay. In this digital grid, you play as a system guardian tasked with fixing broken software architectures by solving Object-Oriented Programming (OOP) and UML (Unified Modeling Language) diagram quizzes while navigating through cyberpunk mazes[cite: 1].

## 🌟 Key Features

- **Cyberpunk Aesthetics:** Immerse yourself in a digital world with a neo-futuristic glassmorphic UI design, dark mode, and dynamic neon-lit background animations[cite: 1].
- **Gamified Learning:** Explore randomized or procedural mazes to find hidden "System Bugs" that trigger software engineering theory and UML diagram challenges[cite: 1].
- **Dynamic HUD System:** Track your progress in real-time with an interactive display showing player name, level, score, and smooth XP level-up animation bars[cite: 1].
- **Immersive Audio Experience:** Retro sci-fi style background music combined with positioned spatial sound effects for laser gates and system interactions[cite: 1].
- **Interactive Info Cards (Modals):** Sleek overlay windows with detailed feedback mechanisms that teach specific UML class diagram relationships, visibility modifiers (`+`, `-`, `#`), and design patterns[cite: 1].

## 🛠️ Tech Stack

This project is a lightweight, pure front-end web application built with a focus on modern web standards and high performance, without using any heavy frameworks or engines (Unity, Unreal, etc.)[cite: 1]:

*   **HTML5 / Canvas API:** Renders the core game loop, character movements, and glowing maze lines seamlessly at a smooth 60 FPS[cite: 1].
*   **CSS3 (Advanced UI & Effects):** Utilizes `Flexbox`, `Grid`, CSS custom variables (`:root`), `backdrop-filter` for glassmorphism, and hardware-accelerated `@keyframes` animations[cite: 1].
*   **Pure Vanilla JavaScript (ES6+):** Handles maze generation, collision detection, quiz logic, level progression, and audio management without external dependencies[cite: 1].
*   **Web Audio API:** Employs embedded data-URI base64 audio assets to ensure lag-free execution of in-game sound effects[cite: 1].

## 🕹️ Controls & Gameplay

- **Movement:** Use the keyboard `Arrow Keys` or `W, A, S, D` keys[cite: 1].
- **Objective:** Navigate through the maze to collect data nodes or approach diagram components[cite: 1].
- **System Barriers (Quizzes):** When a software bug blocks your way, choose the correct UML relationship or OOP concept from the multiple-choice table to earn XP and level up as a system guardian[cite: 1]!
